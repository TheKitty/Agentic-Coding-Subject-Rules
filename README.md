# Agentic-Coding-Subject-Rules

Material to provide Agentic Coding tools specific knowledge for specific tasks.

## Overview

This repository contains rules, guidelines, and reference material designed to give AI-powered agentic coding tools the domain-specific context they need to assist with specialized tasks. By supplying structured subject-matter rules, these tools can produce more accurate, relevant, and project-appropriate code and suggestions.

## Usage

Add or reference the rule files in this repository when configuring your agentic coding tool (e.g., GitHub Copilot, Cursor, or similar). Each rule file targets a specific subject area and provides the context the tool needs to perform well within that domain.
